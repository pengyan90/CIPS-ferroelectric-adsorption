# CIPS Adsorption and Ferroelectric Response (DFT Data)
This repository contains representative density functional theory (DFT) output files associated with the study of molecular adsorption on monolayer CuInP2S6 (CIPS) and its impact on ferroelectric polarization and electronic structure.

## Contents
The repository includes selected Quantum ESPRESSO output files for:
- Pristine monolayer CIPS structures
- Water-covered CIPS configurations
- Adsorbed molecular systems on CIPS

In addition, representative electronic structure data are provided, including:
- Band structure calculations
- Projected density of states (PDOS)

## Notes
- Only representative output files are included to illustrate the key results discussed in the manuscript.
- Input files and full datasets are not included but can be provided upon reasonable request.
- All calculations were performed using density functional theory as described in the manuscript.
## Purpose

This repository is intended to support transparency and reproducibility by providing key computational results underlying the reported findings.
